# Print Method for Reduced Multidesign Objects

Displays a detailed summary of a reduced multidesign object, including
original and reduced dimensions, design variables, and column metadata
if present.

## Usage

``` r
# S3 method for class 'reduced_multidesign'
print(x, ...)
```

## Arguments

- x:

  A reduced_multidesign object

- ...:

  Additional arguments passed to print methods

## Value

Invisibly returns the input object
