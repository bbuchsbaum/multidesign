# Print Method for Multidesign Objects

Displays a detailed summary of a multidesign object, including data
dimensions, design variables, and column metadata if present.

## Usage

``` r
# S3 method for class 'multidesign'
print(x, ...)
```

## Arguments

- x:

  A multidesign object

- ...:

  Additional arguments passed to print methods

## Value

Invisibly returns the input object
