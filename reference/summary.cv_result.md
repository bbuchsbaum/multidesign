# Summary Method for cv_result Objects

Returns a tibble with metric/mean/sd/min/max/median per numeric score
column.

## Usage

``` r
# S3 method for class 'cv_result'
summary(object, ...)
```

## Arguments

- object:

  A cv_result object

- ...:

  Additional arguments (not used)

## Value

A tibble with summary statistics for each metric
