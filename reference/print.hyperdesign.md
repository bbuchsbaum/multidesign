# Print Method for Hyperdesign Objects

Displays a detailed summary of a hyperdesign object, including
information about each block's structure, design variables, and
dimensions.

## Usage

``` r
# S3 method for class 'hyperdesign'
print(x, ...)
```

## Arguments

- x:

  A hyperdesign object

- ...:

  Additional arguments passed to print methods

## Value

Invisibly returns the input object
